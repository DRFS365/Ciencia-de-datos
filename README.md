# 📊 Ciencia de Datos

Repositorio de contenido para estudiar ciencia de datos.

## 🧠 Descripción
Este proyecto recopila materiales para comprender el funcionamiento de los modelos de lenguaje (LLM).  
Está organizado en niveles progresivos, cada uno con código y notas explicativas.

## 📚 Contenido

- 📐 Fundamentos matemáticos  
- 📊 Estadística  
- 🎲 Probabilidad  
- 🤖 Machine Learning  
- 💬 Procesamiento del Lenguaje Natural
